### Добавление JavaScript функциональности

В проект был добавлен файл `js/script.js` с интерактивными функциями для улучшения пользовательского опыта.

#### Реализованные функции:

1. **highlightActivePage()** - Подсветка активной страницы в навигации
   - Автоматически определяет текущую страницу
   - Визуально выделяет соответствующий пункт меню
   - Улучшает ориентацию пользователя на сайте

2. **initSmoothScroll()** - Плавная прокрутка для якорных ссылок
   - Добавляет плавную анимацию при переходе по внутренним ссылкам
   - Улучшает навигацию по странице

3. **initScrollAnimation()** - Анимация появления элементов при прокрутке
   - Использует Intersection Observer API для отслеживания видимости элементов
   - Карточки и секции плавно появляются при прокрутке страницы
   - Создает динамичный и современный интерфейс

4. **initScrollToTop()** - Кнопка "Наверх"
   - Динамически создает кнопку для быстрого возврата наверх страницы
   - Появляется только после прокрутки более чем на 300px
   - Имеет визуальные эффекты при наведении

5. **enhanceCardInteractions()** - Улучшенные эффекты для карточек
   - Добавляет эффект нажатия при клике на карточки
   - Улучшает доступность для навигации с клавиатуры
   - Повышает тактильную обратную связь

6. **updateFooterYear()** - Автоматическое обновление года
   - Динамически обновляет год в футере на текущий
   - Избавляет от необходимости ручного обновления HTML

7. **initFormHandling()** - Обработка и валидация форм
   - Готова для обработки форм обратной связи
   - Валидация обязательных полей
   - Визуальная обратная связь при ошибках

8. **updateVisitCounter()** - Счетчик посещений
   - Использует localStorage для сохранения количества посещений
   - Выводит информацию в консоль браузера

#### Технические детали:

- Все функции инициализируются после полной загрузки DOM
- Использованы современные JavaScript API (Intersection Observer, localStorage)
- Код организован модульно с четким разделением функций
- Добавлена поддержка доступности (ARIA атрибуты)

## Запуск проекта

1. Откройте файл `index.html` в веб-браузере
2. Используйте навигацию для перехода между страницами
3. Попробуйте прокрутить страницу - вы увидите анимации и кнопку "Наверх"
4. Откройте консоль браузера (F12) для просмотра счетчика посещений
